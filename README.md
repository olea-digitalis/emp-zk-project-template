# emp-zk project template

## Install Instructions

1. `git clone https://github.com/olea-digitalis/emp-zk-tutorial.git` then `cd emp-zk-tutorial`
2. `wget https://raw.githubusercontent.com/emp-toolkit/emp-readme/master/scripts/install.py` (source: https://github.com/emp-toolkit/emp-zk )
3. `python[3] install.py --deps --tool --ot --zk`
4. `mkdir data` if there is no `data` directory


## compile & run example file
To compile:
- `cmake . && make`

To run:
- `./run bin/test_basic_zk`
